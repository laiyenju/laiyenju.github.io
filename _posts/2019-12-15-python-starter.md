自學 Python 到現在約 2 個多月，我從零基礎開始學習的路徑：

- 先閱讀[《Python程式設計的樂趣：範例實作與專題研究的20堂程式設計課》](https://www.books.com.tw/products/0010778539)，做完書中的所有案例與練習，建立基本概念。
- 到 [RealPython 規劃的學習路徑](https://realpython.com/learning-paths/)，決定要先在 Python 進階的領域：機器學習 or 網頁開發 or 數據分析。
- 手癢想做小專案前，憑著學生身份申請[ GitHub Student Developer Pack](https://education.github.com/pack)，善用 GitHub 的學生開發者包，有免費的 AWS 雲端服務、code review 或線上課程可多利用。


其實也該多參加社群活動，例如 [Python Taiwan](https://www.facebook.com/groups/pythontw/)、[Taipei.py](http://www.meetup.com/Taipei-py/)、[PyLadies Taiwan](https://www.meetup.com/PyLadiesTW/)，更多在台灣的 Python 社群可[參考](https://tw.pycon.org/2018/zh-hant/about/community/)。
