### Hot Keys
- `F` / `A`   →  draw a new artboard/frame
- `R` →   rectangle mode
- `T` →   text mode
- `Esc` →   exit tool mode
- `Command +` and `Command -` →  zoom in and out
- `Shift + 1` →  zoom and fit to see all artboards/frames
- `Shift + 2` →  zoom and fit to selected item
- `Command (Opt) ]`, `Command (Opt) [` →   arrange items' order
